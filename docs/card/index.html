Cards

Cards groups and displays information related to a product, category or service using components like text, images, badges, etc to grab the user's attention.

Elixir offers the following range of cards - vertical card, horizontal card, card with badge, card with dismiss button, card with overlay text, card with only text, and card with shadow. Verical card, Horizontal card, Card with badge, Card with dismiss button, Card with overlay text, Card with only text, and Card with shadow.
